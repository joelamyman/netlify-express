What is this?

This game allows you to answer questions and create your own e-commerce user interface based on your own needs.

Why does this exist?

This game is part of a student project for my thesis at University of York, UK. The purpose of this game is to understand user preferences and accessibility needs when using user interfaces. The results from this game will help me to understand features of web user interfaces that can be personalised to enhance web accessibility.

What will happen with the information I provide?

No personal information will be captured from this game. There are no cookies or analytical information being tracked either. If you would like to further aid my research, you may wish to provide further contextual information at the end of the game.
